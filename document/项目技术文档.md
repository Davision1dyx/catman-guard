# Catman Guard 项目技术文档

## 一、项目概述

**Catman Guard**（猫猫侠）是一个智能运维助手系统，基于大语言模型构建，提供智能问答、知识库管理、工单中心等核心功能。

### 项目定位
- **目标用户**: 运维人员、技术支持人员
- **核心价值**: 自动化故障排查、智能文档检索、高效工单管理
- **技术栈**: Vue 3 + Spring Boot + PostgreSQL (pgvector)

## 五、数据流分析

### 5.1 智能问答流程

```
前端输入 → ChatController → ChatAgent → ChatModel (LLM) → 流式响应 → 前端渲染
                              │
                              ▼
                    RetrievalAdvisor → 知识库检索
                    TavilyWebSearchTool → 联网搜索
```

**响应数据格式** (SSE流式):
```json
{
  "type": "text|thinking|reference|recommend|error",
  "content": "响应内容"
}
```

### 5.2 文件上传与处理流程

```
文件上传 → FileController → FileService → 存储文件
                                              │
                                              ▼
                                    文件转换 → 文件分片 → 向量生成 → 向量存储
                                    (Convert)  (Chunk)   (Embed)   (VectorDB)
```

**文件状态流转**:
```
UPLOADED (已上传) → CONVERTING (转换中) → CONVERTED (已转换) 
    → CHUNKED (已分片) → VECTOR_STORED (向量存储)
```

### 5.3 工单创建流程

```
前端表单 → IssueController → IssueService → IssueMapper → PostgreSQL
     │                                          │
     ▼                                          ▼
  返回创建结果                           插入issue表
```

## 七、关键技术点

### 7.1 向量数据库集成

项目使用 **PostgreSQL + pgvector** 扩展实现向量存储与检索：
- 支持向量相似度查询
- 文件分片后生成嵌入向量
- 知识库检索基于向量相似度匹配

### 7.2 流式响应 (SSE)

智能对话接口使用 Server-Sent Events 实现实时响应：
```java
@GetMapping("/chat")
public Flux<String> chat(@RequestParam String message, @RequestParam String conversationId) {
    return chatService.chat(conversationId, message);
}
```

### 7.3 对话记忆机制

通过 `ChatMemory` 实现上下文保持：
```java
ChatMemory persistentChatMemory = chatAgent.createPersistentChatMemory(conversationId, 20);
```

## 十、总结

Catman Guard 是一个基于LLM的智能运维助手系统，核心价值在于：

1. **智能问答**: 支持基础对话、联网搜索、知识库检索三种模式
2. **知识库管理**: 文件上传、智能分片、向量存储与检索
3. **工单中心**: 完整的工单CRUD、分配、状态流转管理
4. **值班排班**: 员工排班管理
5. **数据看板**: 运营数据统计与分析

项目采用微服务架构，各模块职责清晰，便于扩展和维护。前端使用Vue 3 + Element Plus构建现代化UI，后端基于Spring Boot框架，数据库采用PostgreSQL配合pgvector扩展实现向量检索功能。
