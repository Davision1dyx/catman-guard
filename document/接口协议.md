# Catman Guard 接口协议文档

## 一、智能问答模块

### 1.1 普通聊天接口

**接口地址**: `GET /processing/catman/conversation/chat/chat`

**请求参数**:

| 参数名 | 类型 | 必填 | 说明 |
| :--- | :--- | :--- | :--- |
| `message` | `string` | 是 | 用户输入的消息内容 |
| `conversationId` | `string` | 是 | 对话ID，用于标识会话 |

**响应格式**: `text/event-stream`（流式响应）

**响应数据结构**:
```json
{
  "type": "text",
  "content": "回答内容"
}
```

### 1.2 网页搜索聊天接口

**接口地址**: `GET /processing/catman/conversation/chat/websearch`

**请求参数**:

| 参数名 | 类型 | 必填 | 说明 |
| :--- | :--- | :--- | :--- |
| `message` | `string` | 是 | 用户输入的消息内容 |
| `conversationId` | `string` | 是 | 对话ID，用于标识会话 |

**响应格式**: `text/event-stream`（流式响应）

### 1.3 检索聊天接口

**接口地址**: `GET /processing/catman/conversation/chat/retrieavl`

**请求参数**:

| 参数名 | 类型 | 必填 | 说明 |
| :--- | :--- | :--- | :--- |
| `message` | `string` | 是 | 用户输入的消息内容 |
| `conversationId` | `string` | 是 | 对话ID，用于标识会话 |

**响应格式**: `text/event-stream`（流式响应）

**响应数据结构**:
```json
{
  "type": "text",
  "content": "回答内容",
  "references": ["参考文档1", "参考文档2"]
}
```

## 二、知识库管理模块

### 2.1 知识库管理

#### 2.1.1 获取知识库列表

**接口地址**: `GET /processing/catman/storage/knowledge/list`

**请求参数**: 无

**响应数据结构**:
```json
{
  "code": 200,
  "data": [
    {
      "knowledgeId": "kb-1",
      "name": "API 网关文档",
      "description": "包含API网关的配置指南",
      "type": "technical",
      "fileCount": 12,
      "chunkCount": 145,
      "createdTime": "2024-01-15T10:00:00",
      "updatedTime": "2024-01-20T15:30:00"
    }
  ]
}
```

#### 2.1.2 创建知识库

**接口地址**: `POST /processing/catman/storage/knowledge/create`

**请求体**:
```json
{
  "name": "知识库名称",
  "description": "知识库描述",
  "type": "technical"
}
```
**响应数据结构**:
```json
{
  "code": 200,
  "data": {
      "knowledgeId": "kb-1"
  }
}
```


#### 2.1.3 更新知识库

**接口地址**: `PUT /processing/catman/storage/knowledge/{id}`

**请求体**:
```json
{
  "name": "更新后的名称",
  "description": "更新后的描述",
  "type": "product"
}
```

**响应数据结构**:
```json
{
  "code": 200
}
```

#### 2.1.4 删除知识库

**接口地址**: `DELETE /processing/catman/storage/knowledge/{id}`

**响应数据结构**:
```json
{
  "code": 200
}
```

#### 2.1.5 知识库类型
| 参数名            | 类型       |
|:---------------|:---------|
| `product`      | `产品文档`   |
| `technical`    | `技术文档`   |
| `operation`    | `运维文档`   |
| `training`     | `培训资料`   |

### 2.2 文件管理

#### 2.2.1 上传文件

**接口地址**: `POST /processing/catman/storage/file/upload`

**请求参数**: `multipart/form-data`

| 参数名               | 类型 | 必填 | 说明     |
|:------------------| :--- | :--- |:-------|
| `file`            | `File` | 是 | 要上传的文件 |
| `title`           | `string` | 是 | 文件标题   |
| `softwareVersion` | `string` | 否 | 软件版本   |
| `feature`         | `string` | 否 | 特性标签   |
| `microservice`    | `string` | 否 | 所属微服务  |
| `description`     | `string` | 否 | 文件描述   |

**响应数据结构**:
```json
{
  "code": 200,
  "data": {
    "fileId": "file-1",
    "url": "http://localhost:1111/file-1",
    "status": "UPLOADED",
    "size": 2048576
  }
}
```

#### 2.2.2 获取文件列表

**接口地址**: `GET /processing/catman/storage/file/list/{knowledgeId}`

**请求参数**:

| 参数名 | 类型 | 必填 | 说明 |
| :--- | :--- | :--- | :--- |
| `search` | `string` | 否 | 搜索关键词 |
| `fileType` | `string` | 否 | 文件类型筛选 |
| `status` | `string` | 否 | 状态筛选 |

**响应数据结构**:
```json
{
  "code": 200,
  "data": [
    {
      "fileId": "file-1",
      "fileTitle": "API Gateway 配置指南",
      "fileName": "api_gateway_config.pdf",
      "version": "v2.3.1",
      "extension": "认证授权",
      "description": "认证授权",
      "microservice": "auth-service",
      "status": "CHUNKED",
      "size": 2048576,
      "createTime": "2024-01-15T10:00:00",
      "updateTime": "2024-01-15T10:00:00"
    }
  ]
}
```

#### 2.2.3 删除文件

**接口地址**: `DELETE /processing/catman/storage/file/{fileId}`

**响应数据结构**:
```json
{
  "code": 200
}
```

### 2.3 文件分片

#### 2.3.0 分片类型

| 参数名            | 类型       |
|:---------------|:---------|
| `fixed_length`      | `固定长度分片` |
| `parent_child`    | `父子分片`   |
| `semantic`    | `语义分片`   |

#### 2.3.1 执行分片

**接口地址**: `POST /processing/catman/storage/chunk/chunk/{fileId}`

**请求体**:
```json
{
  "chunkType": "fixed_length",
  "chunkSize": 1000,
  "chunkOverlap": 100
}
```

**响应数据结构**:
```json
{
  "code": 200,
  "data": {
    "fileId": "file-1",
    "status": "CHUNKED",
    "chunks": [
      {
        "chunkId": "chunk-1-1",
        "content": "分片内容...",
        "size": 1024
      }
    ]
  }
}
```

#### 2.3.2 获取分片信息

**接口地址**: `GET /processing/catman/storage/chunk/list/{fileId}`

**响应数据结构**:
```json
{
  "code": 200,
  "data": {
    "chunks": [
      {
        "chunkId": "chunk-1-1",
        "content": "分片内容...",
        "size": 1024
      }
    ],
    "maxChunkSize": 1500,
    "minChunkSize": 800,
    "avgChunkSize": 1100
  }
}
```

### 2.4 向量库管理

#### 2.4.1 同步向量库

**接口地址**: `POST /processing/catman/storage/embed/sync/{knowledgeId}`

**响应数据结构**:
```json
{
  "code": 200,
  "data": {
    "syncedCount": 145  
  }
}
```

## 三、工单中心模块

### 3.1 工单管理

#### 3.1.0 工单状态

| 参数名            | 类型     |
|:---------------|:-------|
| `ASSIGNED`      | `已派单`  |
| `COMPLETED`    | `处理中` |
| `COMPLETED`    | `完成` |
| `VECTOR_STORED`    | `向量存储` |

工单类型

| 参数名            | 类型     |
|:---------------|:-------|
| `INCIDENT`      | `故障报告`  |
| `REQUEST`    | `服务请求` |
| `PROBLEM`    | `问题分析` |
| `CHANGE`    | `变更请求` |

工单优先级

| 参数名       | 类型   |
|:----------|:-----|
| `HIGH`    | `高`  |
| `MEDIUM` | `中`  |
| `LOW` | `低`  |
| `CRITICAL`  | `紧急` |

## 七、错误码说明

| 错误码 | 说明 |
| :--- | :--- |
| 200 | 成功 |
| 400 | 请求参数错误 |
| 401 | 未认证 |
| 403 | 无权限 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |
